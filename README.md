## 職務経歴（本業）

### **2020/04 - 2023/11**: T株式会社  
#### 事業内容: 大型自動車を中心とした車検整備、点検、修理サービスの提供  
#### 職務: 経営企画

1. **財務体質の改善と黒字転換の実現**
    1. 前年度は営業・経常ともに赤字で推移していたが、原価率と販管費率を見直すことで当期は黒字転換を達成。
    2. 原価率を約−2.3ポイント改善し、粗利率を高水準で維持。さらに、販管費を前年比約8％削減し、損益分岐点を引き下げた。
    3. 結果として、営業利益率は約+5ポイント改善（マイナス→プラス圏へ転換）し、業界平均営業利益率1-3%の中、3.6%を達成。持続的に利益を創出できる構造へと改革。
2. **コスト構造の再設計による収益性向上**
    1. 外注費を約15％削減し、内製化比率を高めることで作業効率と利益率を改善。
    2. 給与手当を約7％圧縮しつつも、生産性を維持・向上させる人員配置を実施。
    3. 結果として、売上高は横ばいながら、営業利益は前年比で20倍超の増益を実現し、粗利率・労働分配率ともに健全化。
3. **経営モニタリング体制の構築と意思決定スピードの向上**
    1. 売上・原価・利益をリアルタイムに把握できる月次モニタリング体制を新設し、データドリブン経営を推進。
    2. 月次売上表や決算書を作成、分析し、サービス別利益率を見直して工賃表を再設計。市場競争力を考慮した適正価格設定を実施。
    3. さらに、分析結果から重点顧客リストを作成し、新規営業戦略を展開することで収益性を強化。

#### 職務： 法人営業 / 営業企画

1. **1.営業戦略の立案・実行による新規顧客開拓**
    1. 市場や競合分析が不十分で、効果的な営業戦略が不足しており、新規顧客の獲得に課題があった。
    2. 市場・競合・自社の分析を基に営業戦略を策定。100社への架電やCEOレター送付を実施。フィールドセールスを通じて5社との新規契約を締結し、新規契約数を年間15件へと拡大。
2. **2.データ分析に基づく既存顧客管理と営業プロセス最適化**
    1. 既存顧客の情報が断片化し、役職ごとのニーズへの対応が不十分で、顧客満足度や取引継続率の向上に課題があった。
    2. 顧客調査とインタビューを設計・実施し、収集したデータを基に顧客ごとの分析レポートを作成。
        1. 顧客ごとにレポートを作成、分析の上で個別対応を徹底し、継続的な取引基盤を強化。
        2. さらに、Notionを営業CRMとして導入し、接触履歴や対応内容を一元管理する仕組みを構築し、営業プロセスの効率化を実現。

#### 職務： バックオフィス、広報

1. **会計業務の自動化とデータ連携の最適化**   
   - Google Apps Scriptを活用し、freee会計からのCSV取込・月次レポート生成を自動化。  
   - これにより、月次締め作業時間を約70％削減し、経理担当者の属人化を解消。  
   - 銀行・仕入・売上データを一元管理し、リアルタイムでの月次損益把握を実現。  

2. **ドキュメント・ワークフローの標準化**  
   - 各種申請書・見積・発注・契約書類をNotionとSlack上で統合。    
   - 書類紛失リスクを解消し、承認スピードを従来比で約2倍に向上。  

3. **データ管理基盤の構築と経営分析の自動化**  
   - 売上・整備履歴・仕入などの業務データをスプレッドシート上で構造化。  
   - Google Drive上のデータを自動収集・集計するETLスクリプトを開発。  
   - KPIダッシュボードを構築し、営業利益率・稼働率・整備単価のリアルタイム監視を可能に。
